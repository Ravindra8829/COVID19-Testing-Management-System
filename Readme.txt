## Overview

This is a web-based COVID-19 Testing Management System developed using PHP and MySQL. The system facilitates the management of COVID-19 testing processes including patient registration, sample collection, testing, and reporting of results.

## Features

- **User Authentication**: Secure login for administrators and lab technicians.
- **Patient Management**: Add, update, view, and delete patient information.
- **Sample Collection**: Record details of collected samples.
- **Test Management**: Enter and update test results.
- **Reporting**: Generate and view test reports.
- **Dashboard**: Overview of the testing statistics.

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Frameworks**: None
## Usage

1. **Login**: Use the provided admin credentials to log in.
2. **Patient Management**: Add new patients and manage existing patient records.
3. **Sample Collection**: Record new sample collections for patients.
4. **Test Management**: Enter test results for collected samples.
5. **Reporting**: Generate and download test reports for patients.
6. **Dashboard**: View the statistics and analytics of the testing data.

How to run the COVID19 Testing Management System Using PHP and MySQL

Download the zip file
Extract the file and copy covid-tms folder
Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
Open PHPMyAdmin (http://localhost/phpmyadmin)
Create a database with name covidtmsdb
Import covidtmsdb.sql file(given inside the zip package in SQL file folder)
Run the script http://localhost/covid-tms
